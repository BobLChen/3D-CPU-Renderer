# Soft3DRenderer

一个非常简单的3D软光栅化小程序，非常适合初学者学习。整个代码只有250多行，但是涵盖了光栅化的基本流程。可以帮助初学者理解OBJ解析加载、MVP矩阵、透视除法、VertexShader和FragmentShader应该在哪一步插入、深度测试的用法以及目的、最终的Color写入需要排序等基本概念。

# 依赖
- CMake 12版本以上

# Build

```bash
cd Soft3DRenderer
mkdir build
cd build
cmake ..
```
